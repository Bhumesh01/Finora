# 💰 Finora – Personal Finance Tracker

Finora is a modern personal finance tracking web application that helps users manage income, expenses, and budgets efficiently. It provides real-time insights into financial activity through a clean and intuitive dashboard.

---

## 🚀 Features

- ➕ Add and manage income & expense transactions  
- 📊 View total balance, income, and expenses  
- 🎯 Set and track category-wise budgets  
- 🔍 Filter transactions by date (month/year)   
- 📈 Smart financial insights and summaries
