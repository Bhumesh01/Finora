# 💰 Finora – Personal Finance Tracker

Finora is a modern personal finance tracking web application built with React, TypeScript, Vite, and Tailwind CSS. It helps users manage income, expenses, and budgets efficiently while displaying real-time financial insights in a clean dashboard layout.

---

## 🚀 Features

- ➕ Add income and expense transactions
- 📊 View total balance, income, and expenses at a glance
- 🎯 Track finances across categories and budgets
- � See spending and income visualized with charts and graphs
- 📉 Compare expenses over time with monthly/annual chart views
- 🔍 Filter transactions by date (month/year)
- 📈 Display summaries for smarter financial decisions

---

## 🧰 Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Chart.js
- ESLint

---

## ⚡ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Bhumesh01/finora.git
   cd finora
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the app:
   - Visit `http://localhost:5173`

---

## 📦 Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the app for production
- `npm run preview` - Preview the production build locally
- `npm run lint` - Run ESLint across the project
